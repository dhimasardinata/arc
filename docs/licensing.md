# Licensing

Arc uses a dual-license style policy: one public copyleft path and one paid
commercial path.

## Public License

The public source license is `AGPL-3.0-only`.

That means you can study, use, modify, and distribute Arc under the GNU Affero
GPL version 3 only. If you distribute modified firmware or run modified
network-accessible services built from Arc, keep the corresponding Arc-covered
source available under the same license terms.

## Paid Commercial Path

The non-AGPL commercial path is not free.

If a product team wants proprietary distribution rights, private source
handling, or permission that avoids AGPLv3-only obligations, it needs a
separate signed paid commercial agreement from the Arc copyright holder.

Cloning, building, embedding, linking, or distributing Arc from this repository
does not create a commercial license.

The intended commercial license floor is at least as strict as an LGPLv3-style
library copyleft for the Arc-covered portions:

- keep Arc notices intact;
- identify Arc in distributed products;
- track changes to Arc itself;
- preserve replacement or inspection rights where the product architecture
  permits it;
- keep patent and anti-circumvention protections no weaker than the
  LGPLv3/GPLv3 family baseline;
- do not turn Arc into a silent permissive dependency.

The commercial license text in this repository is a policy statement, not an
automatic grant. The signed agreement controls the actual commercial rights.

## Practical Decision

Use the AGPL path when your product can satisfy reciprocal source obligations.

Use the paid commercial path when your product needs proprietary distribution or
private modification handling.

See [COMMERCIAL-LICENSE.md](https://github.com/dhimasardinata/arc/blob/main/COMMERCIAL-LICENSE.md)
for the repository-level commercial license path.
